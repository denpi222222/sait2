## Для деплоя на Netlify:
1. Зайти на app.netlify.com
2. New site from Git → GitHub → выбрать sait2
3. Build command: `npm install --legacy-peer-deps && npm run build`
4. Publish directory: `out`
5. Deploy!

## Секреты для GitHub Actions (если нужны):

- NETLIFY_SITE_ID (из site settings)
