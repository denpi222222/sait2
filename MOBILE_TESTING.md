## Как тестировать:

### На компьютере:
1. Откройте Chrome DevTools (F12)
2. Включите режим мобильного устройства (Ctrl+Shift+M)
3. Выберите iPhone или Android устройство
4. Обновите страницу

### На реальном устройстве:

#### iOS (iPhone/iPad):
1. Откройте Safari
2. Перейдите на сайт
3. Нажмите "Connect Wallet"
4. Выберите MetaMask (если установлен) или WalletConnect
5. Для MetaMask: откроется приложение MetaMask
6. Для WalletConnect: отсканируйте QR код

#### Android:
1. Откройте Chrome/MetaMask браузер
2. Перейдите на сайт
3. Процесс аналогичен iOS

#### Telegram:
1. Откройте Telegram
2. Перейдите по ссылке на сайт
3. Используйте WalletConnect для подключения

## Проверьте:

- [x] NFT карточки адаптируются под размер экрана
- [x] Мобильная навигация отображается внизу
- [x] Кнопки достаточно большие для нажатия пальцем
- [x] Модальное окно кошелька открывается снизу
- [x] MetaMask отображается в списке кошельков
- [x] Нет горизонтального скролла
- [x] Текст читаемый на мобильных устройствах
- [x] Социальные иконки слева на десктопе, снизу на мобильных
- [x] NFT карточки в разделе Ping не перекашиваются

## Известные особенности:

1. **MetaMask не виден**: Если установлен Rabby, он может скрывать MetaMask. Отключите Rabby или используйте MetaMask мобильное приложение.

2. **Telegram браузер**: Некоторые функции могут быть ограничены. Рекомендуется использовать WalletConnect.

3. **iOS Safari**: Автоматически увеличивает масштаб при фокусе на input. Это исправлено установкой font-size: 16px.
