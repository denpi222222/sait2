# 🔧 Настройка автоматического деплоя

## 📋 Пошаговая инструкция

### 1. 🚀 Деплой на GitHub
Запустите батник для отправки проекта на GitHub:
```bash
./deploy-to-github.bat
```

### 2. 🌐 Настройка Netlify

1. **Зайдите на [netlify.com](https://netlify.com)**
2. **Войдите через GitHub аккаунт**
3. **Нажмите "New site from Git"**
4. **Выберите репозиторий: `denpi222222/sait2`**
5. **Настройки сборки:**
   - Build command: `npm run build && npm run export`
   - Publish directory: `out`
   - Node version: `20`

### 3. 🔑 Настройка секретов GitHub

Для автоматического деплоя нужно добавить токены в GitHub:

1. **Получите токены Netlify:**
   - Зайдите в Netlify → User settings → Applications
   - Создайте новый Access Token
   - Скопируйте Site ID из настроек сайта

2. **Добавьте в GitHub Secrets:**
   - Зайдите в репозиторий → Settings → Secrets and variables → Actions
   - Добавьте:
     - `NETLIFY_AUTH_TOKEN` - ваш Access Token
     - `NETLIFY_SITE_ID` - ID сайта

### 4. ✅ Проверка работы

После настройки:
- Каждый `git push` в ветку `main` запустит автоматический деплой
- Статус можно увидеть во вкладке "Actions" в GitHub
- Сайт обновится автоматически на Netlify
